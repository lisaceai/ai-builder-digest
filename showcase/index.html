01 / 系统架构
端到端自动化工作流
从推文抓取到邮件推送，全链路通过 GitHub Actions 实现定时自动化。 每日定时触发 Apify 爬取指定 Builder 的推文，经 AI 摘要后存入 Pinecone 向量数据库， 同时发送邮件摘要，支持 Web 端实时问答和趋势分析。
02

Builder 清单管理

通过 Web 界面轻松管理你关注的 AI Builder 列表。 添加或删除 Builder 后，系统自动同步到 GitHub 仓库的配置文件， 下次定时任务将自动抓取新增 Builder 的推文数据。

实时增删 自动同步 GitHub 配置即生效
03

每日定时邮件摘要

GitHub Actions 每日定时执行，自动将 AI 生成的推文摘要 以精美的 HTML 邮件形式推送到你的邮箱。 无需主动查看，打开邮件即可掌握 AI 圈最新动态。

GitHub Actions 定时 HTML 邮件模板 AI 摘要
04 / 核心功能
RAG 智能问答 + 来源追溯
基于 Pinecone 向量数据库的检索增强生成（RAG）。 输入任意问题，系统从历史推文中检索最相关的内容，结合 AI 生成精准回答， 并附带原文来源链接，每一条回答都可追溯验证。
提问并获得 AI 回答
来源追溯与原文链接
05

AI 趋势分析

基于时间维度对推文数据进行聚合分析，自动识别热点话题与技术趋势。 支持按时间范围筛选（24小时/3天/7天/30天）， 还可针对单个 Builder 深度分析其关注方向和讨论模式。

时间范围筛选 热点话题识别 Builder 个体分析
